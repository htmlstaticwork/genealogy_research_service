Genealogy Research Service

Welcome to the complete documentation guide for the Genealogy Research Service website. This professional template is designed for ancestry research, historical tracking, and family tree archival services.

Installation Guide

1

Download & Extract

Download the template package and extract it to your local machine. You will see folders for CSS, JS, and all core HTML pages.

2

Folder Structure

The project is organized logically for easy maintenance:

/genealogy-service/
├── css/ (All stylesheets)
├── js/ (Interactivity scripts)
├── documentation/ (This guide)
├── images/ (Asset library)
├── index.html (Main Landing Page)
├── home2.html (Variant Landing Page)
├── about.html (Company Information)
├── services.html (Service Packages)
├── contact.html (Booking & Inquiries)
├── dashboard.html (User Portal)
└── pricing.html (Cost Breakdown)
3

Local Setup (VS Code)

Open the folder in VS Code. For the best experience, install the Live Server extension to preview changes in real-time.

4

Hosting & Deployment

This is a static HTML/CSS/JS site. You can host it on:

  • GitHub Pages
  • Vercel / Netlify
  • CPanel via FTP

Pro Tip: Always keep a backup of the original `css/` files before making heavy customizations.

Customization Guide

Colors (CSS Variables)

You can change the entire theme by modifying the CSS variables in the root of your stylesheet:

:root {
  --primary: #38bdf8;  /* Change this for brand color */
  --bg-dark: #0f172a;  /* Main background */
  --accent: #ec4899;   /* Secondary accent */
}

Typography

The site uses Inter and Outfit from Google Fonts. To change fonts, update the link in the `` of your HTML files and update the font-family in CSS.

/* BEFORE */

font-family: 'Inter', sans-serif;

/* AFTER */

font-family: 'Roboto', sans-serif;

Image Placeholders

All images are located in the `/images` folder. Ensure your new images maintain the original aspect ratios for the best layout results.

Recommended Sizes:

  • Hero Images: 1920x1080px
  • Feature Tiles: 800x600px
  • Team Avatars: 400x400px

Troubleshooting FAQ


Q: Layout is breaking on small screens?

A: Ensure you have the meta viewport tag in your head and that no fixed-width CSS elements were added.

Q: Images are not loading?

A: Verify the path in your code. Are images in /images folder? Remember, file names are case-sensitive.
